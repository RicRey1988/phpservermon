# PHP Server Monitor v4.3.2-hs

Esta versión culmina la migración visual nativa de PHP Server Monitor HS a HOPE UI y mantiene compatibilidad con PHP 8.5 o superior.

## Interfaz HOPE UI nativa

- Eliminado completamente `src/templates/default/static/css/hs-monitor.css`.
- La aplicación carga exclusivamente `hope-ui.min.css`, `dark.min.css` y `customizer.min.css` incluidos con HOPE UI.
- Migradas las vistas de Estado, Estadísticas, Servidores, Registro, Historial, Configuración, Sistema, Usuarios, Perfil, Notificaciones, autenticación e instalador a cards, grids, formularios, badges, listas y utilidades responsivas nativas.
- Eliminadas clases Bootstrap 4 y clases antiguas que dependían del stylesheet personalizado.

## Barra superior y navegación

- Alineación uniforme de los iconos de tema, configuración y notificaciones.
- El icono de modo claro u oscuro conserva contraste y centrado en ambos esquemas.
- Buscador reconstruido con el `input-group search-input` original de HOPE UI.
- Flecha de colapso ubicada sobre el borde exterior y rotada mediante el comportamiento `sidebar-mini` de la plantilla.
- Eliminado el bloque duplicado de usuario, Perfil y Salir del menú lateral; las acciones permanecen en el menú superior.
- El menú móvil abre y cierra correctamente, actualiza `aria-expanded` y se cierra al navegar, pulsar Escape o hacer clic fuera.

## PWA y actualización

- Caché estática rotada a `psm-static-4.3.2-hs-r2` para retirar el CSS eliminado en instalaciones existentes.
- Las respuestas privadas o dinámicas siguen sin almacenarse en caché.
- El release incluye el ZIP de producción, manifiesto canónico y firma RSA-SHA256 requeridos por el actualizador HS.

## Mejoras incluidas

- Sesiones persistentes con rotación segura y cierre de sesión revocable.
- Corrección del generador VAPID y normalización del contacto `mailto:`.
- Logo del sistema y foto de perfil procesados de forma segura a WebP.
- Diagnósticos PHP y del sistema agrupados sin exponer credenciales.
- Estados online, offline, warning y paused más claros y actualizados inmediatamente.

## Verificación

- Suite completa de PHPUnit ejecutada con PHP 8.5.
- Análisis estático con PHPStan y validación estricta de Composer.
- Comprobación sintáctica de los runtimes JavaScript y del service worker.
- Contratos responsivos para 360, 390, 768, 1024, 1366 y 1600 píxeles en esquemas claro y oscuro.
- Auditoría que impide reintroducir `hs-monitor.css`, clases Bootstrap 4 o el contrato móvil `sidebar-open`.

## Actualización manual

Conserva `src/config/config.php`, `logs/`, imágenes subidas y datos de runtime. Después de reemplazar los archivos ejecuta:

```bash
php bin/psm migrate
php bin/psm health
```

Requisitos mínimos: PHP 8.5+, MySQL o MariaDB y las extensiones declaradas en `composer.json`.
