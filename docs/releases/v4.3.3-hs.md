# PHP Server Monitor v4.3.3-hs

Esta versión publica la migración completa a HOPE UI nativo bajo un número superior a `4.3.2-hs`, de modo que las instalaciones existentes puedan recibirla mediante el actualizador firmado integrado.

## Interfaz HOPE UI nativa

- `hs-monitor.css` permanece eliminado por completo.
- La aplicación carga únicamente `hope-ui.min.css`, `dark.min.css` y `customizer.min.css` incluidos con HOPE UI.
- Estado, Estadísticas, Servidores, Registro, Historial, Configuración, Sistema, Usuarios, Perfil, Notificaciones, autenticación e instalador usan cards, grids, formularios, badges, listas y utilidades responsivas nativas.
- Se retiraron las clases antiguas de presentación y los contratos heredados de Bootstrap 4.

## Barra superior y navegación

- Iconos de tema, configuración y notificaciones centrados y alineados de manera uniforme.
- El selector claro/oscuro conserva contraste y tamaño en ambos esquemas.
- Buscador reconstruido con el `input-group search-input` original de HOPE UI.
- Flecha del sidebar colocada y rotada mediante el estado nativo `sidebar-mini`.
- Eliminado el bloque duplicado de usuario, Perfil y Salir del menú lateral.
- Menú móvil funcional con sincronización de `aria-expanded` y cierre al navegar, pulsar Escape o hacer clic fuera.

## PWA y actualización

- Caché estática rotada a `psm-static-4.3.3-hs-r1`.
- Assets HOPE UI y JavaScript versionados con `4.3.3-hs` para evitar recursos obsoletos.
- Las respuestas privadas y dinámicas continúan excluidas de la caché.
- El release contiene el ZIP de producción, el manifiesto canónico SHA-256 y la firma RSA-SHA256 exigidos por el actualizador HS.

## Verificación

- PHP 8.5.
- Validación estricta de Composer y comprobación de requisitos de plataforma.
- Auditoría de dependencias.
- Suite completa de PHPUnit.
- PHPStan.
- Validación sintáctica de los runtimes JavaScript y del service worker.
- Instalación de producción sin dependencias de desarrollo.

## Actualización

Desde una instalación `4.3.2-hs`, abre **Sistema → Actualizar** y aplica `4.3.3-hs`. El actualizador preserva la configuración privada y valida repositorio, versión, manifiesto, hash, firma y contenido del ZIP antes de instalar.

En una actualización manual deben conservarse `src/config/config.php`, `logs/`, imágenes subidas y datos de runtime. Después de reemplazar los archivos ejecuta:

```bash
php bin/psm migrate
php bin/psm health
```

Requisitos mínimos: PHP 8.5+, MySQL o MariaDB y las extensiones declaradas en `composer.json`.
