# PHP Server Monitor v4.3.4-hs

Esta versión conserva íntegramente la migración visual HOPE UI publicada en `v4.3.3-hs` y corrige la compatibilidad del actualizador con el nombre canónico actual del repositorio de Hosting Supremo.

## Actualizador firmado

- Acepta los assets publicados bajo `RicRey1988/phpservermon-Redesigned-by-hostingsupremo`.
- Mantiene compatibilidad con el identificador anterior `RicRey1988/phpservermon`, que GitHub continúa redirigiendo.
- Conserva las validaciones de digest SHA-256, manifiesto canónico, firma RSA y contenido seguro del ZIP.

## Despliegue y recuperación

- El workflow de respaldo detecta `src/config/config.php` o `config.php` sin mostrar credenciales.
- Las asignaciones extraídas desde PHP usan saltos de línea reales antes de ejecutar `mariadb-dump`.
- Migración, salud y rollback de aplicación/base de datos continúan fallando de forma segura.

## Interfaz

- Se mantienen alineados los iconos de tema, configuración y notificaciones.
- La búsqueda conserva el `input-group` nativo de HOPE UI en claro y oscuro.
- El sidebar usa la flecha y rotación nativas, no duplica la sesión y abre/cierra correctamente en móvil.
- La auditoría autenticada permite indicar un ejecutable Chromium/Chrome mediante `PSM_BROWSER_EXECUTABLE`.

## Actualización

La instalación valida los tres assets firmados de `v4.3.4-hs`. Después de actualizar, ejecuta:

```bash
php bin/psm migrate
php bin/psm health
```
