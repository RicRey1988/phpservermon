# PHP Server Monitor v4.3.5-hs

Esta revisión completa la migración responsiva a HOPE UI corrigiendo la
interacción del menú lateral en dispositivos móviles.

## Correcciones

- El contenido interno del menú HOPE UI queda recortado mediante la utilidad
  nativa `overflow-hidden` mientras el panel está fuera de pantalla.
- El logotipo oculto ya no intercepta el botón de apertura de la barra superior.
- El menú conserva la apertura, el cierre al pulsar fuera, el cierre con Escape
  y los estados ARIA sincronizados.
- La imagen decorativa de cabecera usa la utilidad nativa `start-0`, eliminando
  16 píxeles de desbordamiento horizontal en pantallas estrechas; el shell
  también garantiza esa clase al iniciar.
- La caché PWA cambia a `psm-static-4.3.5-hs-r1` para entregar el arreglo sin
  conservar recursos anteriores.
- La auditoría responsiva admite `PSM_APP_SHELL_OVERRIDE` para validar un
  candidato local contra un entorno autenticado antes de publicarlo.
- La prueba de cierre exterior pulsa ahora el borde derecho visible, fuera del
  panel abierto.
- Los service workers se bloquean durante la auditoría de código para impedir
  que una caché anterior sustituya al candidato evaluado.
- El diagnóstico ignora descendientes ya recortados por un panel cerrado, sin
  relajar la comprobación del ancho desplazable del documento.
- La auditoría espera la transición de HOPE UI y la sincronización ARIA, y
  reconoce regiones de desplazamiento horizontal intencional.
- Los nodos SVG auxiliares transparentes o sin tamaño no se tratan como
  desbordamientos visuales.

## Comprobación posterior

```bash
php bin/psm migrate
php bin/psm health
```

El release incluye el ZIP de producción, su manifiesto SHA-256 canónico y la
firma RSA-SHA256 separada que valida el actualizador integrado.
